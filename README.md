# system-dashboard
ターミナルで使用Linux向けのダッシュボードです。
開発中です。

## 機能
- CPU 使用率(CPU Usage)
- メモリ使用量(Memory Usage)
- 稼働時間(Uptime)
- ストレージ情報(Storage)
- CPU 温度(CPU Temperature)

## 実行方法
```bash
pip3 install -r requirements.txt
            or
pip3 install --break-system-packages -r requirements.txt

python3 system-dashboard.py
